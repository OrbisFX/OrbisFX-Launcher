Please do not add to, edit, or replace any of the shader or texture files included with GShade.

Additional shaders (.fx, .fxh) should be placed in:
 - ?:\Users\Public\GShade Custom Shaders\Shaders\

Additional compute shaders (.fx, .fxh, not compatible with DirectX 9/10) should be placed in:
 - ?:\Users\Public\GShade Custom Shaders\ComputeShaders\

Additional textures (.jpg, .jpeg, .png, .dds) should be placed in:
 - ?:\Users\Public\GShade Custom Shaders\Textures\

Presets (.ini) should be placed in either the Custom folder or a new folder inside of:
 - ?:\Path\To\Your\Game\gshade-presets\

Remember to preserve any folder structure provided by the shader or preset creators when adding to the above folders.

If you're unsure how to navigate to the GShade Custom Shaders folder, type the following in your Start Menu search and hit Enter:
 - %Public%\GShade Custom Shaders
